User's 2K monitor rotated portrait — preferred HTML output width is 1000px.

**Why:** 1440px was too wide, 1100px was still "a touch too wide", 1000px was accepted without complaint.

**How to apply:** Any HTML report, brief, or visual output intended for the portrait monitor should default to `width: 1000px` with matching `<meta name="viewport" content="width=1000">`.
