Peter has slightly diminished vision. For all visual work:

- Prefer serif typefaces over sans-serif where legibility is a factor — serifs guide the eye at smaller sizes
- High contrast is non-negotiable (already enforced by gold-on-navy palette)
- Don't make text elements or labels small "for elegance" — legibility beats refinement
- When sizing elements, test at the actual rendered size, not the design size
