Peter has a `www` K8s object in his homelab cluster that can host personal web pages. Use this as an alternative or supplement to GitHub Pages when deploying static HTML sites, portfolio pages, or project showcases.

**Why:** Gives him full control over hosting without depending on GitHub Pages. Useful for pages he wants on his own domain.

**How to apply:** When deploying a static site (like the ai-cost-containment showcase), offer to deploy to the www K8s object in addition to or instead of GitHub Pages. Ask for the target URL/hostname if not known.
