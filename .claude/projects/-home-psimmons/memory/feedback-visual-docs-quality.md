Magazine-quality HTML documentation (full-color, SVG diagrams, WWII military aesthetic) hit the right bar — user wanted to share with human colleagues as a showcase of what agents produce.

**Why:** The visual quality of documentation reflects on the whole Generals system. If it looks polished and professional, it builds credibility.

**How to apply:** When creating visual docs, guides, or reference materials, invest in the design — inline SVGs, color-coded elements, proper typography, print-friendly layout. Self-contained HTML (no external deps). The WWII military palette (olive #4A5320, khaki #C3B091, navy #0F172A, brass #D4A574) tested well. Aim for "would hang on a wall" quality.
