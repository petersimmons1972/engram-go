Single-source technical claims are worthless. Always require 2+ independent sources before treating a claim as verified.

**Why:** Anecdotal evidence from one Reddit post or one HN comment doesn't constitute reliable intelligence. Only when multiple independent practitioners report the same issue/behavior does it become actionable.

**How to apply:** When extracting technical claims from research data, discard anything that only appears in one source. Cross-validate across Reddit, HN, GitHub, StackExchange, etc. The threshold for "verified" is 2+ independent sources reporting the same claim.
